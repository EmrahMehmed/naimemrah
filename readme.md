# GIT HUB CODES

## STAGE

- dosyalari stage etmek icin command
- `git add {dosya adi}`
- Hepsini bir anda secmek icin
- `git add .`

## COMMIT

- Dosyalari COMMIT etmek icin
- `git commit -m "{commit mesaji}"`

## PUSH

- Dosya degisikliklerini GitHub a eklemek icin
- ` git push`
- Dosyalari cekmek icin
- `git pull`
